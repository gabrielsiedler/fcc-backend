Timestamp API

Request

Method

GET

URL

API/{timestamp}
{timestamp}: UNIX (ex. "1475204400") or Natural (ex. "September 30, 2016") date

Response

Status

200

Response

{
    "unix": {unixDate},
    "natural": {naturalDate}
}
{unixDate}: Date in Unix format (ex. 1475204400) {naturalDate}: Date in Natural format (ex. December 12, 2016)

Status

200

Response

{
    "unix": null,
    "natural": null
}
* Returned when data is not a valid date in the Natural or Unix format.

Example

Requisition

API/1450137600

Response

{
    "unix": "1450137600",
    "natural": "December 14, 2015"
}

Requisition

API/invalid

Response

{
    "unix": null,
    "natural": null
}
